# unistyle

unistyle is a browser extension that converts selected text into various styles, based on PlainStyle (by shapoco).

## Installation
1. Download or clone this repository.
2. Run `npm install` to fetch the dependency (plainstyle.js).
    - Or, download https://www.shapoco.net/plainstyle/plainstyle_rev2.js and place it in the same directory as `background.js`.
3. Open `chrome://extensions/` in your browser.
4. Enable **Developer mode**.
5. Click **Load unpacked** and select the directory of this project.
